# BC Expert Consultation

Get expert advice from BC Code Intelligence specialists.

## Usage

```
/bc-expert "How do I implement custom posting routines?"
```

## What It Does

Spawns the **bc-expert** agent to consult BC specialists and document findings.

## Output

- `.dev/expert-[topic].md` - Consultation results with recommendations and code examples

## When to Use

- Need expert BC/AL advice
- Complex architecture decisions
- Performance optimization questions
- Security concerns
- Best practice guidance
- Debugging BC-specific issues

## Available Specialists

The agent will route your question to the appropriate specialist:
- Dean Debug - Debugging
- Alex Architect - Architecture
- Pat Performance - Performance
- Roger Reviewer - Code review
- Sam Security - Security
- And more...
