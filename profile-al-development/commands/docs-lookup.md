# Microsoft Docs Lookup

Search and retrieve official Microsoft Learn documentation.

## Usage

```
/docs-lookup "table extension syntax"
```

## What It Does

Spawns the **docs-lookup** agent to search Microsoft Docs and document findings.

## Output

- `.dev/docs-[topic].md` - Official documentation, API references, code examples

## When to Use

- Need official AL syntax reference
- Want to see base app API documentation
- Looking for breaking changes in BC versions
- Need official best practices
- Want code examples from Microsoft

## Example Queries

- "table extension patterns"
- "event subscriber syntax"
- "Customer table API reference"
- "BC v23 breaking changes"
